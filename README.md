# Python-Actividad-2

Resolución del punto 10 de la práctica 2 de Python, con Jupyter Notebook y estructura src/notebooks.
Este proyecto resuelve una competencia de cocina donde se evalúan distintas rondas y se calcula una tabla de posiciones final.


# Estructura:

- `src/funciones.py`: contiene las funciones principales del programa.
- `notebooks/ejercicio10.ipynb`: ejecución del programa y visualización de resultados.


# Requisitos:

- Python 3.x
- Jupyter Notebook (opcional, para ejecutar el notebook).


# Cómo ejecutar el programa:

1. Clonar el repositorio: git clone https://github.com/lautaro70/Python-Actividad-2.git
2. Abrir el proyecto en tu editor.
3. Ejecutar el notebook:
- Abrir `notebooks/ejercicio10.ipynb`
- Ejecutar todas las celdas (Run All).


# Descripción:

El programa:
- Calcula el puntaje de cada cocinero por ronda
- Determina el ganador de cada ronda
- Actualiza una tabla acumulada
- Muestra la tabla ordenada por puntaje total


# Salida:

Se imprime una tabla con:
- Puntaje total
- Rondas ganadas
- Mejor puntaje
- Promedio por ronda
